Adding Overlap Leniency

Since the subject AND query guide are getting truncated, the total maximum overlap is going to be double our coordinate adjustment.

L1 -> +/- 1 from each end; 10% of guide L2 -> +/- 2 from each end; 20% of guide L3 -> +/- 3 from each end; 30% of guide L4 -> +/- 4 from each end; 40% of guide L5 -> +/- 5 from each end; 50% of guide

GDO200L1 <- GDO200 %>% mutate(startL = start+1, endL = end-1)
GDO200L2 <- GDO200 %>% mutate(startL = start+2, endL = end-2)
